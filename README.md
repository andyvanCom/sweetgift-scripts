# SweetGift Scripts

Модульная система JavaScript для сайта **SweetGift.ru**.

Проект построен по принципу единого загрузчика (**Loader**) и независимых модулей. В `<head>` сайта подключается только `sweetgift-loader.js`, который загружает необходимые скрипты в зависимости от текущей страницы.

---

# Архитектура

```
Tilda
   │
   ▼
sweetgift-loader.js
   │
   ▼
sweetgift-manifest.json
   │
   ├── sweetgift-share.js
   ├── sweetgift-product-badges.js
   ├── sweetgift-live-popup.js
   ├── sweetgift-article-stats.js
   ├── sweetgift-product-top-lists.js
   ├── sweetgift-product-analytics.js
   ├── sweetgift-order-analytics.js
   └── sweetgift-core.js (в разработке)
```

# Структура проекта

## sweetgift-loader.js

Главный загрузчик проекта.

Функции:

* загрузка manifest
* определение текущей страницы
* подключение нужных модулей
* предотвращение повторной загрузки

---

## sweetgift-manifest.json

Конфигурация всех модулей проекта.

Для каждого модуля хранится:

* название
* путь к файлу
* версия
* включён / выключен
* страницы подключения

---

## sweetgift-share.js

Единый модуль "Поделиться".

Используется:

* в карточках товаров
* в статьях

Поддерживает:

* Telegram
* VK
* WhatsApp
* Web Share API
* Копирование ссылки

---

## sweetgift-product-badges.js

Выводит бейдж активности товара на основании реальной статистики.

Используются события:

* просмотры
* добавление в корзину
* избранное
* переходы
* репосты
* общий рейтинг популярности

---

## sweetgift-live-popup.js

Попап последних действий покупателей.

Используется для повышения доверия.

---

## sweetgift-article-stats.js

Статистика статей.

Выводит:

* дату обновления
* время чтения
* просмотры
* лайки
* кнопку "Поделиться"

Работает только на:

```
/stati/*
```

---

## sweetgift-product-top-lists.js

Выводит в карточке товара блок

> 🏆 Товар входит в подборки SweetGift

Данные загружаются из Supabase.

---

## sweetgift-product-analytics.js

Записывает поведение покупателей.

События:

* просмотр карточки
* переход из каталога
* добавление в корзину
* избранное
* поделиться

---

## sweetgift-order-analytics.js

Записывает оформленные заказы.

Планируется сохранять:

* товары
* стоимость заказа
* стоимость доставки
* дата доставки
* интервал доставки
* "В подарок"
* контакты получателя

---

## sweetgift-core.js

**Статус:** в разработке.

Будет выполнять функции:

* единое подключение Supabase
* очередь записи событий
* кэширование
* общие функции
* fingerprint пользователя
* повторное использование данных всеми модулями

После внедрения остальные модули перестанут самостоятельно подключать Supabase.

---

# Аналитика

Используется Supabase.

Основные таблицы:

```
product_events
products_stats
orders
top_lists
top_list_items
```

Основные RPC:

```
track_product_event()
track_order()
get_product_activity_badges()
get_product_top_lists()
get_category_product_insights()
get_public_top_lists_page_period()
refresh_top_lists_period()
```
