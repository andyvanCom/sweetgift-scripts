# Цель проекта

Преобразовать SweetGift.ru из интернет-магазина на Tilda
в интеллектуальную платформу съедобных подарков,
которая самостоятельно анализирует данные, рекомендует товары,
строит SEO-страницы и помогает покупателям выбрать подарок.

# Стратегия

Каждая новая функция должна использовать
единое хранилище данных Supabase.

Новые модули не должны обращаться напрямую к Tilda.

Источником данных становится собственная платформа SweetGift.

Tilda остается системой управления витриной,
а бизнес-логика постепенно полностью переносится в SweetGift Platform.
